## ⚙️ Deployment

Choose one of the following (all free):

### Option 1: GitHub Pages (easiest)

1. **Create a new repository** on GitHub named `productivity-tools-hub`.
2. **Upload all files** from this folder to the repository.
3. Go to **Settings → Pages**, select `main` branch as source, and save.
4. Your site will be live at `https://<your-username>.github.io/productivity-tools-hub/` within minutes.

### Option 2: Netlify (drag‑and‑drop)

1. Go to [netlify.com](https://netlify.com) and sign up.
2. Drag the entire folder onto the Netlify dashboard.
3. Your site gets a `.netlify.app` domain instantly.

### Option 3: Vercel

1. Go to [vercel.com](https://vercel.com) and sign up.
2. Import this folder as a new project.
3. Click deploy – done.

## 🤑 Monetization Setup

To start earning, sign up for these free affiliate programs (takes ~20 minutes):

1. **Amazon Associates** – for physical products (chairs, monitors, etc.)
   - Apply at [affiliate-program.amazon.com](https://affiliate-program.amazon.com)
   - Once approved, replace `amazon.com/dp/B08XYZ` placeholder links with your own affiliate tags.

2. **ShareASale** – for software (Notion, Zoom, etc.)
   - Apply at [shareasale.com](https://shareasale.com)
   - Search for merchants like “Notion”, “Zoom”, “Logitech” and get unique links.

3. **ConvertKit** – for email newsletter (free up to 1,000 subscribers)
   - Sign up at [convertkit.com](https://convertkit.com)
   - Create a form and replace the `action` URL in `index.html` with your own.

4. **Google Analytics** – to track traffic
   - Create a property at [analytics.google.com](https://analytics.google.com)
   - Replace `G-XXXXXX` in `index.html` with your measurement ID.

## ✏️ Adding More Content

1. Write new articles in `articles-md/` as Markdown files with front matter.
2. Run `node build.js` to generate HTML.
3. Run `node build-final.js` to update the homepage article grid.

## 🔧 Customization

- **Site name:** Change “Productivity Tools Hub” in `index.html`.
- **Colors:** Edit `css/style.css` variables.
- **Favicon:** Add `favicon.ico` to the root.
- **Logo:** Replace the 🚀 emoji with an image.
